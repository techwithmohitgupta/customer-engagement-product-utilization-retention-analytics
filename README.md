## 📌 Project Overview

This project is an interactive **Customer Engagement & Product Utilization Analytics Dashboard** built to support banking customer retention strategy.

The dashboard analyzes customer engagement behavior, product usage depth, high-value disengagement risk, credit card stickiness, churn exposure, and relationship strength scoring across a customer-level banking churn dataset.

It is designed to help banking, customer success, CRM, and analytics teams answer critical retention questions:

* Do active customers retain better than inactive customers?
* Which engagement segment contributes the most to churn?
* Does product depth improve retention?
* Which product-count groups show churn risk?
* Are high-balance customers always loyal?
* Which high-value customers are disengaged and at risk?
* Can retention strength scoring separate sticky customers from high-risk customers?
* Which customer segments should receive priority retention actions?

---

## 🎯 Business Problem

Banks often analyze churn using broad demographic or financial indicators, but customer retention is strongly influenced by engagement behavior, product adoption, relationship depth, and product stickiness.

A customer may have a high balance or high salary but still be at churn risk if they are inactive, under-engaged, or weakly connected to the bank’s product ecosystem.

Without a centralized retention analytics dashboard, it becomes difficult to monitor:

* Active vs inactive customer retention
* Product depth impact on churn
* Single-product vs multi-product customer behavior
* High-value disengaged customers
* Credit card stickiness
* Relationship strength score movement
* Critical retention-risk customer groups
* Segment-level retention strategy priorities

This project solves that problem by converting customer churn data into a dynamic Streamlit dashboard with KPIs, charts, heatmaps, bubble maps, insight sections, and retention strategy recommendations.

## 🧩 Dashboard Modules

The dashboard is divided into four major analytical modules.

---

### 1️⃣ Engagement vs Churn Overview

This module evaluates the relationship between customer engagement behavior and churn.

**Key Features**

* Customers analyzed
* Retained customers
* Churned customers
* Overall retention rate
* Retention gap
* Inactive churn lift
* Highest churn segment
* Best retention segment
* Engagement outcome split
* Churn contribution share
* Engagement × retention tier churn risk matrix

**Business Value**

This section helps identify whether active customers retain better than inactive customers and which engagement segment should receive first campaign priority.

---

### 2️⃣ Product Utilization Impact Analysis

This module analyzes how product count and product depth influence retention.

**Key Features**

* Average product count
* Single-product churn rate
* Multi-product retention rate
* Strongest product count
* Best retention product count
* Highest churn product count
* Multi vs single retention gap
* Product count impact chart
* Single vs multi-product retention split
* Product relationship strength bubble map
* Product depth × retention tier risk matrix

**Business Value**

This section helps evaluate whether product adoption improves loyalty, where product bundling should focus, and whether very high product load creates churn risk.

---

### 3️⃣ High-Value Disengaged Customer Detector

This module identifies financially valuable customers who may still carry churn risk because of weak engagement or low relationship strength.

**Key Features**

* High-value customer count
* High-value disengaged count
* Disengagement rate
* Balance at risk
* Average disengaged balance
* Average disengaged salary
* Highest salary-risk group
* High-value customer segment mix
* Balance exposure by segment
* Salary-balance mismatch bubble map
* High-value engagement × retention tier risk matrix

**Business Value**

This section helps detect silent churn risk among premium customers and supports targeted high-value recovery campaigns.

---

### 4️⃣ Retention Strength Scoring Panels

This module converts engagement, product depth, and relationship signals into retention-strength intelligence.

**Key Features**

* Average retention strength score
* Average relationship strength index
* Strongest retention tier
* Highest risk tier
* Retention strength score distribution
* Relationship strength distribution by retention tier
* Relationship strength vs retention strength map
* Retention tier churn risk ranking

**Business Value**

This section helps prioritize customers by risk level and supports campaign sequencing based on retention strength tiers.

---

## 🧮 KPI Logic

```text
Engagement Retention Ratio =
Active Member Retention Rate / Inactive Member Retention Rate

Product Depth Index =
Average NumOfProducts among Retained Customers /
Average NumOfProducts Overall × 100

High-Balance Disengagement Rate =
High Value Disengaged Customers /
High Value Customers × 100

Credit Card Stickiness Score =
Retained Credit Card Customers /
Total Credit Card Customers × 100

Relationship Strength Index =
Average Relationship_Strength_Index
```

## ▶️ How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/techwithmohitgupta/customer-engagement-product-utilization-retention-analytics.git
```

### 2. Navigate to the Project Folder

```bash
cd customer-engagement-product-utilization-retention-analytics
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

For Windows:

```bash
venv\Scripts\activate
```

For macOS/Linux:

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the Streamlit App

```bash
streamlit run app/app.py
```

## 🏁 Conclusion

This project demonstrates how customer engagement, product utilization, high-value customer risk, and retention strength scoring can be combined into a practical banking retention analytics framework.

The dashboard helps move retention strategy from generic churn monitoring to targeted, evidence-based decision-making by identifying inactive churn contributors, optimal product-depth segments, high-value disengaged customers, and critical retention-risk tiers.

By combining Python, Pandas, NumPy, Plotly, Streamlit, and published research documentation, this project provides an end-to-end applied data analytics portfolio case study for banking customer retention strategy.
